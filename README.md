Basic pomodoro timer example.

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

## Quick Start
```sh
cd react-pomodoro
npm install
npm start
```
Open `http://localhost:3000` to view it in your browser.
